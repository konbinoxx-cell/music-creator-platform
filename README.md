# 🎵 简易音乐创作平台

一个基于Web的交互式音乐创作工具，让每个人都能轻松创作音乐！

## ✨ 功能特色

- 🎼 **可视化五线谱** - 点击即可添加音符
- 🎹 **智能伴奏生成** - 基于旋律自动生成和声
- 🎚️ **多种音乐风格** - 欢快、平静、神秘、活力
- ⏺️ **实时录音** - 录制你的音乐作品
- 💾 **音频导出** - 导出为webm格式

## 🚀 快速开始

1. **直接使用**：
   - 点击 `index.html` 文件，用浏览器打开即可使用

2. **在线体验**：
   - 将项目部署到GitHub Pages或任何静态托管服务

## 🎮 使用方法

### 基础操作
1. **添加音符**：点击五线谱区域添加音符
2. **试听音符**：点击音符按钮或已添加的音符
3. **删除音符**：双击音符删除
4. **播放乐谱**：点击"播放乐谱"聆听创作

### 智能创作
1. **创作旋律**：在五线谱上点击添加几个音符
2. **分析旋律**：点击"分析旋律"让系统理解你的创作
3. **生成伴奏**：点击"智能生成伴奏"获得完整音乐

### 高级功能
- **扩展乐谱**：点击"添加小节线"创作更长作品
- **风格切换**：选择不同音乐风格改变整体感觉
- **录音导出**：录制并下载你的音乐作品

## 🛠️ 技术栈

- **前端**：纯HTML5 + CSS3 + JavaScript
- **音频**：Tone.js Web Audio框架
- **响应式**：支持桌面和移动设备

## 🌟 特色亮点

- **中央C标识**：加粗显示中央C线，便于定位
- **可扩展五线谱**：支持多小节创作
- **智能和声**：基于用户旋律生成合适的伴奏
- **无需安装**：单文件运行，开箱即用
